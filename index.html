memory that works like memory

cairn stores facts the way your brain does. with context, confidence, and decay.

01 / temporal
bi-temporal storage
store facts with valid time and system time. query what was true on any date. history isn't erased, it's superseded.

graph traversal

1
remember
save a fact to memory. the agent calls this as an mcp tool. no extra inference, no extra cost.
$ cairn remember "fact"
2
recall
find facts in memory. keywords find entry points, graph traversal walks the subgraph, ranking by confidence and recency.
$ cairn recall "query"
3
forget
remove a fact. or let it decay. low-confidence facts get tombstoned. user-stated facts never auto-forget.
$ cairn forget "fact"
stop building agents that forget. give them memory.
