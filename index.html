three commands. that's it.

no config files. no json editing. no api keys. remember, recall, forget.

1

Remember

store any fact with a confidence score. cairn extracts the key automatically. no llm needed.

$ cairn remember "user likes vim"
2

Recall

query by meaning, not exact match. cairn finds the closest facts and ranks by confidence and recency.

$ cairn recall "text editor pref"
3

Forget

prune stale facts that dropped below threshold. or forget a specific key. memory stays clean.

$ cairn forget --stale

agents leave cairns to mark what they learned

stop building agents that forget everything at session end. give them memory that persists.
